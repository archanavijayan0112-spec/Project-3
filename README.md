# Project-3
Mental Wellness Web Application
Mental Wellness Web Application is a comprehensive wellness platform designed to promote mental health through AI-driven assistance and personalized tracking tools. The application includes an AI chatbot, mood and sleep monitoring, music therapy recommendations, motivational content, and therapist discovery services. Built using Python Flask and modern web technologies, the platform provides a responsive user experience across web and mobile devices.

***Key Features***

AI-powered wellness chatbot

Mood tracking and analytics

Sleep monitoring system

Therapist finder module

Music therapy recommendations

Motivational content delivery

Responsive web and mobile interface

User wellness dashboard

Python

Flask

HTML5

CSS3
